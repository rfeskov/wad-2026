# WAD — Web Application Development

Materials for the **WAD** (web application development) course: **lecture code samples** and the **homework specification**.

## Lecture examples

Examples from classes live in numbered folders (`1-lesson/`, `2-lesson/`, …). Each file is a small standalone demo (async, WSGI/ASGI, Starlette, FastAPI, DI, OAuth samples, etc.). Open the folder for the lesson you need and run or inspect the scripts as shown in class.

## Homework

The assignment (stack, architecture, LLM chat, auth, deadlines, PDF report, and deliverables) is described here:

**[HOMEWORK_SPEC.md](HOMEWORK_SPEC.md)**

## Local LLM snippets (optional)

In the repo root you can also find minimal **`llama-cpp-python`** examples (`main.py`, `streaming.py`) and a placeholder **`model.gguf`** path for local CPU inference, as referenced in the homework spec.
